Install Manual
  • Checkout code with following command: svn checkout http://esnippet.googlecode.com/svn/trunk/ esnippet-read-only
  • Modify settings.properties file to set correct configurations
  • Import database tables and import sample data. Please import 001_initial_db.sql into MySQL
  • Maven package: mvn package
  • Deploy war file as ROOT application
  • Start your browser to visit UI
